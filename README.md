# 🫀 Heart Stroke Risk Prediction System

A Machine Learning-powered healthcare application that predicts an individual's stroke risk using clinical and lifestyle factors. The system provides real-time risk assessment along with preventive healthcare recommendations, doctor guidance, diet suggestions, and medication reminders.

## 🚀 Features

* Stroke risk prediction using Machine Learning
* Risk categorization: **Low**, **Medium**, and **High**
* Real-time prediction through a Flask web application
* Personalized health recommendations
* Doctor consultation suggestions
* Diet and lifestyle guidance
* Medication reminder functionality
* Secure user authentication

## 🧠 Machine Learning Models

The system utilizes ensemble learning techniques to improve prediction accuracy:

* Random Forest Classifier
* Gradient Boosting Classifier

## 📊 Input Parameters

The model analyzes multiple health-related attributes, including:

* Age
* BMI (Body Mass Index)
* Average Glucose Level
* Hypertension
* Heart Disease History
* Smoking Status
* Lifestyle Factors

## 🔄 System Workflow

1. User enters health information.
2. Data is preprocessed and validated.
3. Machine Learning model predicts stroke risk.
4. Risk level is displayed instantly.
5. Healthcare recommendations are generated.
